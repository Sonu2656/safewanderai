# 🌍 SafeWander

> **AI-powered safety companion for travellers, students, and solo explorers.**
> Generate a live safety brief for any destination, find nearby help, and carry your emergency info anywhere — even offline.

🔗 **Live demo:** [safewanderai.lovable.app](https://safewanderai.lovable.app)

![SafeWander hero](src/assets/hero-globe.jpg)

---

## ✨ What it does

SafeWander turns a destination name into a **personalised, real-time safety brief** in seconds — built for the moments that actually matter when you land somewhere new.

### Core features

- 🧠 **AI Trip Brief** — Safety score (0–100), risk snapshot, advisory status, money tips, cultural etiquette, and a tailored packing list, all generated by Google Gemini.
- 💬 **Trip Assistant** — Streaming AI chat that knows your destination, weather, and traveller profile. Ask follow-ups like *"Is the metro safe at night?"* or *"What's a fair taxi price?"*.
- 🗺️ **Nearby help map** — Hospitals, police, pharmacies, and embassies within 3 km, powered by OpenStreetMap + Overpass.
- 📞 **Local emergency numbers** — Police / ambulance / fire / general per country, with one-tap dialling.
- ☀️ **Live weather** — Current conditions and "feels like" temperature for your destination (Open-Meteo).
- 🆘 **Whisper SOS** — A floating panic button with live GPS, one-tap emergency call, copy-to-share location, and text-to-speech of local emergency phrases.
- 🪪 **Offline Safety Card** *(unique)* — Print or download a wallet-sized card with your blood type, allergies, emergency contact, local emergency numbers, and translated emergency phrases. Works fully offline.
- 👤 **Traveller profile** — Save your home country, emergency contact, blood type, allergies, and medical notes once; SafeWander uses them across every trip and on your safety card.

## 🚀 Run locally

> Requires Node 18+ and a package manager (`bun`, `pnpm`, `npm`, or `yarn`).

```bash
git clone https://github.com/<your-username>/safewander.git
cd safewander
bun install            # or: npm install
bun dev                # or: npm run dev
```

The app expects the following environment variables (auto-managed in Lovable Cloud — copy from `.env`):

```
VITE_SUPABASE_URL=
VITE_SUPABASE_PUBLISHABLE_KEY=
VITE_SUPABASE_PROJECT_ID=
```

Open <http://localhost:5173> and create an account.

## 🔐 Privacy

- Health & emergency-contact details stay on your device (`localStorage`).
- Trip briefs are saved to your account and are **only readable by you** thanks to Row-Level Security.
- No third-party tracking.
